Detailed Description

It's a base class for all Grid, Detail and Forms models. Most works for Forms does this class.

It response for Forms API: getPage, insertItem, updateItem, getEditItem, deleteItem. Also this class expose basic common method which used by all Forms like dictionaries, table sharing etc
